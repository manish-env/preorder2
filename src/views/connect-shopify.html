How to get your API Access Token:

  1. 1Go to your Shopify Admin Dashboard
  2. 2Navigate to Apps → App and sales channel settings
  3. 3Click "Develop apps" → "Create an app"
  4. 4Give your app a name (e.g., "Preorder Dashboard")
  5. 5Go to "Configuration" tab
  6. 6Under "Admin API access scopes", enable: read_products, write_products, read_orders, write_orders
  7. 7Click "Save" then "Install app"
  8. 8Copy the "Admin API access token"
